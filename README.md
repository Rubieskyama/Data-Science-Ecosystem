# IBM Data Science Professional Certificate

This repository contains the labs, coding assignments, and final capstone project completed as part of the **IBM Data Science Professional Certificate** on Coursera. This collection demonstrates the end-to-end data science lifecycle, from data acquisition and preprocessing to modeling and deployment.



![Data Science Lifecycle](https://analyticstraininghub.com/wp-content/uploads/2022/10/data-science-800x671.png)


## 🎓 Course Curriculum
The certification comprises 10 modules covering foundational and advanced data science concepts:

| Module | Course Title | Key Focus |
| :--- | :--- | :--- |
| 1-3 | Intro & Methodology | Data Science basics and industry workflows. |
| 4-5 | Python for Data Science | Data structures, APIs, and Web Scraping. |
| 6 | Databases & SQL | Relational databases and SQL for data analysis. |
| 7 | Data Analysis | Exploratory Data Analysis (EDA) with Pandas. |
| 8 | Data Visualization | Matplotlib, Seaborn, Folium, and Plotly. |
| 9 | Machine Learning | Regression, Classification, Clustering, Recommender Systems. |
| 10 | Applied Capstone | End-to-end project solving a real-world problem. |

## 🛠 Technical Stack
* **Languages:** Python (Pandas, NumPy, Scikit-Learn)
* **Databases:** SQL (SQLite, PostgreSQL, MySQL)
* **Visualization:** Matplotlib, Seaborn, Folium, Plotly, Dash
* **Machine Learning:** Regression, Classification, Clustering, Neural Networks
* **Tools:** Jupyter Notebooks, IBM Watson Studio, Git/GitHub

## 🚀 Applied Data Science Capstone
The final project focuses on predicting the success of Falcon 9 first-stage landings to estimate launch costs. By applying binary classification models to historical launch data, I developed a predictive framework that enables competitors to better understand SpaceX’s cost-efficiency through stage reusability. 
**View the Capstone Project here:** [Capstone Project Repository](https://github.com/Rubieskyama/SpaceX-Falcon-9-First-Stage-Landing-Prediction)
